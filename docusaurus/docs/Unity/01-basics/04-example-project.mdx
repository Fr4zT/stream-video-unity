Stream's Video SDK for Unity contains an example project showing how to implement video streaming and receiving between multiple call participants.

Once you've [installed](./basics/install/) the SDK you can import the `Sample` project.

### How to import the sample project
1. Open Unity's package manager by selecting from the top menu `Window -> Package Manager`
2. In the top of the `Package Manager` windows select `Packages` to `In Project` (By default it's `Unity Registry`)
3. Select the `Stream Video & Audio Chat SDK` package
4. In the `Samples` dropdown, click `Import` button that is next to `Video & Audio chat sample`
